# IntelliPrice - Laptop Price Predictor 

A Machine Learning web application that predicts laptop prices based on specifications such as RAM, processor, storage, and brand.

##  Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML/CSS

##  Project Features
- Trained ML regression model
- Flask web interface
- Real-time price prediction
- Data preprocessing and feature engineering

##  Project Structure
- model/ → Trained ML model (predictor.pickle)
- website/ → Flask web application
